GIMP AT SIGGRAPH 2000
GIMP -- Gnu Image Manipulation Program is a freely distributed program that allows users to manipulate and create images with its powerful collection of tools and plug-ins. The GIMP is an open source project that different people and companies are contributing.

The GIMP has joined forces with special effects world to transform the GIMP into a program that can be used in film production. The first  result of this effort was the Hollywood branch, this 16 bit GIMP is used today in production at Rhythm & Hues. Currently we are working together on a new engine that has been designed to tackle problems related to film production, such as 16 bits, different data types, different white points, and color models.
