GEGL -- Gimp 'E' Graphical Library --- is an image processing library based on GtkObjects. This library has been designed initially to handle various image processing tasks needed in Gimp, but should be useful  to other apps as well. It is designed to treat different color models and data types  in a general way.  It includes a way of creating chains (or trees) of image operations and evaluating these in a demand-driven way.  Classes for point, area, geometric and statistics operations are provided, as well as color conversion ops for converting images between color models.  Memory management and image classes are included. It also includes a method for writing generic image processing routines (Generic Image Language) and generating specific data type and colormodel code (Codegen) from this.
