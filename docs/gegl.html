Gegl -- GIMP 'e' Graphical Library --- is the new engine for the GIMP 2.0. This engine has been designed to handle various data types and color models. The GIMP -- Gnu Image Manipulation Program -- is an free application that allows user to do image retouching, image compositing, and image authoring. Apart form the 16 bit hollywood branch of the GIMP has only supported 8 bits. With Gegl, the GIMP will become an excellent tool for all costums needs.
