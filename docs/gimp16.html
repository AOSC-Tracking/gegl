GIMP's Film Version
The GIMP is moving towards becoming a tool for film industry. It's 16 bit developement version is currently being used in film production by various special effects houses and there is a current movement to re-architect the GIMP to handle the new demands. 
GEGL  is a new image processing library that has been designed to support different data types and color models. This library will become the new engine for GIMP 2.0. 
There is a 16 bit version of the GIMP that is currently being used in film production. To get a copy  of this GIMP either get it via ftp or via cvs (check out the HOLLYWOOD version of the gimp and gimp-data-min).  Here are some notes by Steve MacPherson of how to install this version of GIMP on a SGI.
